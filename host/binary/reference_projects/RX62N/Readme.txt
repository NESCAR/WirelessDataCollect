3. Supported modes
  WLAN only

4. Directory structure
  RX62N        : This folder contains the sample project for RX62N 
 
5. Steps to build 
  RX62N        : 
         1. Open the project in using micro controller supported IDE (for
         RX62N use high performance embedded workshop (HEW) IDE). 
         2. Set configurations in host/binary/apis/wlan/ref_apps/include/rsi_config.h
         3. Build the application.
         4. Run the application.
